# RURBANIVE Plant Species Database

A plant species database homepage developed as part of the RURBANIVE Project.

This page presents a searchable and interactive collection of plant species used in ecological restoration on Milos Island following quarrying activities. It supports exploration of indigenous species adapted to local conditions and highlights species of ecological, cultural, and practical value.

## Project Context

This homepage is part of:

RUE#2  
Ecosystem and Biodiversity Restoration

Part of the RURBANIVE Project

Lead Partners: Agricultural University of Athens (AUA) and IMERYS

## Overview

The Plant Species Database homepage provides an accessible interface for browsing plant species relevant to restoration activities on Milos Island.

It includes:

- a searchable plant card interface
- filtering by edible species
- filtering by bee-friendly species
- random plant selection
- dynamic loading of plant data from a CSV database
- animated navigation toolbar consistent with the RURBANIVE homepage design

The database focuses primarily on indigenous species suited to the environmental conditions of Milos and relevant to ecosystem recovery and restoration practice.

## Features

- Search by scientific name, Greek name, or family
- Filter plants by edible value
- Filter plants by bee-plant value
- Random plant selection
- Responsive card-based layout
- Animated blob-style navigation menu
- Dynamic CSV-based plant data loading

- HTML5
